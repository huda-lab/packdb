Dynamic Constraints

Intermediate

Use scalar subqueries and CASE expressions to compute constraint bounds dynamically.

Scalar Subquery CASE Expression COUNT Simulation

Query

SQL 
SELECT
    item_id,
    name,
    category,
    value,
    x as selected
FROM Items
DECIDE x
SUCH THAT
    x IS BINARY,
    -- Scalar subquery: dynamic max items from config table
    SUM(x) <= (SELECT max_items FROM Config WHERE id = 1),
    -- CASE expression: count category A items (COUNT simulation)
    SUM(x * CASE WHEN category = 'A' THEN 1 ELSE 0 END) >= 2,
    -- CASE expression: limit category B items
    SUM(x * CASE WHEN category = 'B' THEN 1 ELSE 0 END) <= 3,
    -- Budget constraint with dynamic bound
    SUM(x * value) <= (SELECT budget FROM Config WHERE id = 1)
MAXIMIZE SUM(x * value);

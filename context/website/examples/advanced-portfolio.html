Portfolio Optimization

Advanced

Select assets to maximize expected return while respecting risk limits and diversification rules.

Scalar Subquery CASE Expression Complex Coefficients

Query

SQL 
SELECT
    ticker,
    name,
    sector,
    expected_return,
    risk_score,
    min_investment,
    x as selected
FROM Assets
DECIDE x
SUCH THAT
    x IS BINARY,

    -- Budget constraint: total investment within budget
    -- Uses subquery to get budget from config table
    SUM(x * min_investment) <= (SELECT value FROM Config WHERE name = 'budget'),

    -- Risk constraint: total risk budget limited to 300 points
    SUM(x * risk_score) <= 300,

    -- Diversification: at least 5 different assets
    SUM(x) >= 5,

    -- Sector limits: no more than 2 assets per sector
    -- Technology sector
    SUM(x * CASE WHEN sector = 'Technology' THEN 1 ELSE 0 END) <= 2,
    -- Healthcare sector
    SUM(x * CASE WHEN sector = 'Healthcare' THEN 1 ELSE 0 END) <= 2,
    -- Finance sector
    SUM(x * CASE WHEN sector = 'Finance' THEN 1 ELSE 0 END) <= 2,
    -- Energy sector
    SUM(x * CASE WHEN sector = 'Energy' THEN 1 ELSE 0 END) <= 2,
    -- Consumer sector
    SUM(x * CASE WHEN sector = 'Consumer' THEN 1 ELSE 0 END) <= 2

MAXIMIZE SUM(x * expected_return * min_investment / 100);

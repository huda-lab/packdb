Multi-Variable Assignment

Intermediate Coming Soon

Assign workers to tasks with multiple decision variables per row and equality constraints.

Multiple Decision Variables Equality Constraints PER Clause

Coming Soon: This example uses the PER clause for grouped constraints. PER is a planned feature and is not yet available in the current build. The syntax shown is the intended API.

Query

SQL 
SELECT
    worker_id,
    task_id,
    skill_match,
    hours,
    x as assigned,
    y as hours_allocated
FROM Assignments
DECIDE x, y
SUCH THAT
    x IS BINARY,
    y IS INTEGER,
    -- Equality constraint: if assigned, allocate exact hours needed
    y = x * hours,
    -- Each worker can be assigned to at most 1 task
    SUM(x) PER worker_id <= 1,
    -- Each task must be assigned to exactly 1 worker
    SUM(x) PER task_id = 1
MAXIMIZE SUM(x * skill_match);

Diet Planning

Intermediate

Select meals to minimize cost while meeting nutritional requirements for protein, carbs, and fat.

Binary Variables Multiple Constraints Lower+Upper Bounds MINIMIZE

Sample Data

SQL 
CREATE TABLE Meals (
    id INTEGER,
    name VARCHAR,
    protein INTEGER,
    carbs INTEGER,
    fat INTEGER,
    calories INTEGER,
    cost DOUBLE
);

INSERT INTO Meals VALUES
    (1, 'Grilled Chicken',    35, 20, 12, 420, 8.50),
    (2, 'Caesar Salad',       10, 15,  8, 220, 6.00),
    (3, 'Beef Stew',          30, 25, 18, 480, 9.00),
    (4, 'Veggie Stir Fry',    15, 40, 10, 350, 5.50),
    (5, 'Fish Tacos',         28, 30, 14, 400, 7.50),
    (6, 'Greek Yogurt Bowl',  20, 30,  8, 280, 4.00),
    (7, 'Turkey Sandwich',    25, 35, 10, 380, 5.00),
    (8, 'Lentil Soup',        18, 45,  6, 320, 4.50);

Query

SQL 
SELECT
    id,
    name,
    protein,
    carbs,
    fat,
    calories,
    cost,
    x as selected
FROM Meals
DECIDE x
SUCH THAT
    x IS BINARY,
    -- Minimum protein: 50g
    SUM(x * protein) >= 50,
    -- Minimum carbs: 100g
    SUM(x * carbs) >= 100,
    -- Maximum fat: 50g
    SUM(x * fat) <= 50,
    -- Maximum calories: 1500
    SUM(x * calories) <= 1500
MINIMIZE SUM(x * cost);

Expected Output

id name protein carbs fat calories cost selected
6 Greek Yogurt Bowl 20 30 8 280 4.00 1
7 Turkey Sandwich 25 35 10 380 5.00 1
8 Lentil Soup 18 45 6 320 4.50 1
